## Quick Start

Choose your preferred language:

```typescript
// Node.js
import { Client } from '@autonomi/client';
const client = new Client();
```

```python
# Python
from autonomi import Client
client = Client()
```

```rust
// Rust
use autonomi::Client;
let client = Client::new()?;
```
